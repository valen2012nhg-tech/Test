# Addon: DRGBase Nexbot Cazador

Este addon añade un **nexbot para Garry's Mod** basado en **DRGBase**.

## Comportamiento
- Busca al jugador humano más cercano.
- Lo persigue continuamente usando pathfinding de NextBot.
- Cuando está en rango, ataca con daño cuerpo a cuerpo hasta matarlo.

## Instalación
1. Instala [DRGBase](https://steamcommunity.com/sharedfiles/filedetails/?id=1560118657).
2. Copia esta carpeta en `garrysmod/addons/`.
3. Inicia el juego y spawnea `DRG Cazador` desde el menú de entidades.

## Entidad
- Clase: `drg_nexbot_cazador`
- Categoría: `DRGBase Nextbots`
